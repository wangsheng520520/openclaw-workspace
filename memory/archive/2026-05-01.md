### Bonjour mDNS Discovery 启用

**时间**: 2026-05-01 22:46~22:58

**配置**:
```json
"discovery": {
  "mdns": {
    "mode": "minimal"
  }
}
```

**验证结果** (22:58):
```
+ eth0 IPv4 BF-202511271408 (OpenClaw)    _openclaw-gw._tcp    local
  address = [192.168.1.5]
  port = [18789]
  txt = ["transport=gateway" "displayName=BF-202511271408" 
         "lanHost=openclaw.local" "gatewayPort=18789" "role=gateway"]
```

**前置条件**:
- WSL 已启用镜像模式 (networkingMode=mirrored)，获得真实局域网 IP 192.168.1.5
- avahi-utils v0.8 已预装 (包含 avahi-browse 命令)

**历史背景** (MEMORY.md 04-26):
- 旧版 bonjour/CIAO mDNS advertiser 在 WSL2 Hyper-V NAT 下导致 Gateway 崩溃
- 已在 v2026.4.23 中移除，仅保留内置 bonjour-discovery 功能
- 镜像模式 + minimal mode = bonjour discovery 正常工作
