### v2026.5.6 沙盒重大变化

**问题**: 升级后 agent 完全被锁死
- `capabilities=none`
- 无网络（curl/wget/gateway 均不可达）
- 无 host FS 访问（read/write 均报 path escapes sandbox）
- 工作区只读
- node/npm/docker/openclaw CLI 全部不可用
- 退化为"纯思维实体"——只能分析推理+session 管理，无法操作系统

**根因**: 上一轮安全加固时设置的 `sandbox.mode="all"`，v2026.5.6 的沙盒实现比 v2026.5.2 严格得多

**解决**: `sandbox.mode="off"` 全局关闭沙盒，所有能力恢复

### 关键教训
- **sandbox.mode 升级敏感**: 同一配置在不同 OpenClaw 版本下行为差异巨大
- **安全加固需验证兼容性**: 在升级前设置严格安全策略，升级后可能彻底锁死
- **MEMORY.md 截断循环**: Gateway 重启会回注 compact 后的内容，压缩效果不稳定；提高上限比反复清理更可靠
- **飞书 groupPolicy=allowlist 需要 channel 级 groupAllowFrom**: 只设 account 级不够，channel 级为空会导致所有群消息沉默丢弃
