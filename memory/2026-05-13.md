## 根因分析 (03:15 CST)

### 发现：memory-core 插件被禁用

**检查结果**:
```
│ Memory       │ memory   │          │          │ Runs a bounded blocking memory sub-agent... │          │
│ @openclaw/   │ memory-  │ openclaw │ disabled │ stock:memory-core/index.js                  │ 2026.5.7 │
```

- `memory-core` (内置记忆核心): **disabled**
- `memory-lancedb` (LanceDB 向量存储): **enabled**
- `memory-wiki` (Obsidian Wiki 集成): **enabled**

### 分析

Memory Dreaming Promotion cron (`412d9a19`) 执行时：
1. 调用 `memory-core` 进行梦境处理
2. `memory-core` 被禁用 → 进程无法正常运行
3. Cron 调度器可能认为执行成功（进程启动）但实际无效果
4. 或者调度器在调用时发现 memory-core 不可用而跳过

### 其他受影响的 Cron

| Cron ID | 功能 | 状态 |
|---------|------|------|
| 412d9a19 | Memory Dreaming Promotion | **disabled** (memory-core) |
| 013cf5c6 | 知识图谱每日更新 v3 | ok |
| 470d911a | 每日记忆提炼 | ok |
| 53de5b3f | SESSION-STATE 新鲜度检查 | ok |

### 结论

**根因**: `memory-core` 插件被禁用，导致 Memory Dreaming Promotion cron 无法正常工作，梦境晋升自 05-07 后停止。

**修复方案**:
```bash
openclaw plugins enable memory-core
openclaw gateway restart
```

**或通过 config 启用**:
```json
{
  "plugins": {
    "entries": {
      "memory-core": { "enabled": true }
    }
  }
}
```
