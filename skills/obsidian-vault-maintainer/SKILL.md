# Workflow

## Workflow

### 微信公众号文章导入双链流程
1. 内容提取：使用web_fetch获取公众号文章完整正文，过滤广告、小程序提示、扫码引导等冗余内容
2. 关联匹配：用grep搜索知识库中同主题的现有笔记，筛选3-7篇最相关的关联对象
3. 新笔记创建：
   - 添加标准Frontmatter：标题、作者、来源URL、发布日期、标签、创建时间、内容摘要
   - 保留原文核心内容，末尾添加「相关笔记」区块，指向所有关联笔记
   - 按内容类型归类到对应目录（灵感库/内容工厂/已发布归档）
4. 索引更新：在对应主题的MOC/知识库索引页的合适分类下添加新笔记链接
5. 反向链接同步：选择2-3篇最核心的关联笔记，在其「相关笔记」区块添加新笔记的反向引用
6. 输出交付：告知用户笔记位置、标签列表、建立的双向链接统计、文章核心内容摘要
