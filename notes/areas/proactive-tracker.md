```bash
# 每 2 小时检查一次，进程数 > 20 则 pkill 清理 + systemctl restart
# 已有清理逻辑，需确保持续运行
```

**预估效果**：消除手动干预需求
**预估复杂度**：simple
**优先级**：🔴 高 — **04-28 再次发生，04-23 cron 可能未有效运行**
